Simple FDTD Wave propagation in MATLAB

Single-file codes in MATLAB for acoustic and elastic wave propagation in time domain using finite-difference approximation(FDTD). As simple formulation and implementation as possible. Just for fun and to have all these codes in one place.

Acoustic

Only P waves propagate in acoustic medium. Parametrized by compressional wave velocity Vp only.

Acoustic 2D isotropic
Acoustic 3D isotropic

Elastic

P and S waves propagate in elastic medium. It is parametrized by compressional Vp and shear Vs wave velocities.

Elastic 2D isotropic
Elastic 3D isotropic

Bonus:

ELASTIC TI

Transverse isotropy is typically observed in layered media. In this case wave propagation goes with different velocities in the in-plane and cross-plane directions. It is parametrized by four parameters in 2D and by five in 3D.

Elastic 2D TI
Elastic 3D VTI
